# CFRU structural-donor crosswalk

Radical Red is built on the open-source Complete FireRed Upgrade (CFRU) engine (`github.com/Skeli789/Complete-Fire-Red-Upgrade`), unlike Unbound where the closest available donor was an approximating third-party fork. This doc records whether CFRU's public source can be pattern-matched directly against Radical Red's compiled binary to shortcut RE (skipping blind text/pointer anchor-hunting for some systems), or whether Radical Red's own years of feature work have drifted the binary too far from a stock CFRU compile for that to pay off.

Time-boxed to 1-2 sessions early in Phase 1. Record the outcome either way so it isn't re-attempted if it doesn't pay off.

Not yet attempted.
